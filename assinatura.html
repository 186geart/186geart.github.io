Criar Assinatura Institucional

Para criar a sua assinatura institucional, realize os seguintes passos:

  • Preencha os campos abaixo corretamente, usando letra maiúscula apenas na primeira letra de cada termo do nome e função.
  • Copie o código gerado na última caixa.
  • Após copiar o código, siga as instruções:
    1. Deste tutorial para inserir no Gmail.
    2. Deste Tutorial para inserir no Thunderbird.

Preencha o formulário para gerar sua assinatura

Como usar:

Para o Gmail:

  1. Preencha o formulário acima e clique em "Gerar Assinatura".
  2. Selecione a assinatura gerada com o mouse.
  3. Copie (Ctrl+C) e cole (Ctrl+V) no Gmail.

Para o Thunderbird:

  1. Preencha o formulário acima e clique em "Gerar Assinatura".
  2. Clique no botão "Copiar para Thunderbird".
  3. Cole o código HTML diretamente no Thunderbird.
